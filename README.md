### 干 (如何跑起来)
> 拉取项目到本地
```bash
git clone https://github.com/Wxh16144/love.git
```
> 切换到目录,并安装依赖
```bash
cd love && npm i
```
> 启动 (嘴角微微上扬🐶)
```bash
npm start
```

### 📦打包 
> 自己~~折腾~~研究去吧;[parcel](https://parceljs.org)打包工具
